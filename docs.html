UI2TOOL (JS UI Component Library)

Thư viện thành phần bổ sung UI đơn giản, mở rộng cho extension và web app. Cung cấp Tooltip, Toast, Alert (modal) không cần cấu hình phức tạp (nhúng là dùng).

UI2TOOL là gì?
Thư viện giao diện đơn giản, dễ mở rộng cho tiện ích trình duyệt và web app, hỗ trợ Tooltip, Toast, Alert (modal)... mà không cần cấu hình phức tạp.

Cài đặt

Nhúng script UI2TOOL vào HTML của bạn: 

<script src="UI2TOOL.js"></script>

Sau khi nhúng, bạn có thể sử dụng UI2TOOL ở bất kỳ đâu trong code mà không cần cài đặt thêm.

Cách sử dụng

Sử dụng toàn cục

Gọi các phương thức UI2TOOL trực tiếp không cần khởi tạo: 

// Tooltip
UI2TOOL.Tooltip(element, "Nội dung tooltip", "top");
UI2TOOL.Tooltip.showTooltip(element, "Nội dung tooltip", "right");
UI2TOOL.Tooltip.hide();

// Toast
UI2TOOL.Toast("Đây là thông báo toast");  // Cú pháp rút gọn
UI2TOOL.Toast.Show("Thông báo toast với cú pháp đầy đủ");
UI2TOOL.Toast("Thông báo thành công", { type: "success" });
UI2TOOL.Toast("Thông báo ở góc trên bên phải", { position: "top-right" });

// Alert
UI2TOOL.Alert("Nội dung thông báo");  // Cú pháp rút gọn
UI2TOOL.Alert.Show("Nội dung thông báo", "Tiêu đề");

Sử dụng instance riêng biệt

Tạo nhiều instance UI cho các phần khác nhau trong ứng dụng: 

// Khởi tạo các instance riêng biệt
const UI1_Left = new UI2TOOL('ui-left');     // Tạo container với id="ui-left"
const UI2_Center = new UI2TOOL('ui-center'); // Tạo container với id="ui-center"
const UI3_Right = new UI2TOOL('ui-right');   // Tạo container với id="ui-right"

// CSS tùy chỉnh vị trí (tùy chọn)
// #ui-left .u2t-toast-container { left: 1rem; right: auto; }
// #ui-center .u2t-toast-container { left: 50%; transform: translateX(-50%); right: auto; }
// #ui-right .u2t-toast-container { right: 1rem; left: auto; }

// Sử dụng instance UI1_Left
UI1_Left.Toast("Thông báo ở UI bên trái");
UI1_Left.Alert("Cảnh báo ở UI bên trái");
UI1_Left.Tooltip(element, "Tooltip ở UI bên trái", "top");

// Sử dụng instance UI2_Center
UI2_Center.Toast("Thông báo ở UI trung tâm");
UI2_Center.Alert("Cảnh báo ở UI trung tâm");
UI2_Center.Tooltip(element, "Tooltip ở UI trung tâm", "bottom");

// Sử dụng instance UI3_Right
UI3_Right.Toast("Thông báo ở UI bên phải");
UI3_Right.Alert("Cảnh báo ở UI bên phải");
UI3_Right.Tooltip(element, "Tooltip ở UI bên phải", "right");

Tooltip

Tạo tooltip cho bất kỳ phần tử nào với vị trí linh hoạt.

Sử dụng cơ bản 

// Gắn tooltip khi hover
UI2TOOL.Tooltip.setTooltip(element, "Nội dung tooltip", "top");

// Hiển thị tooltip ngay lập tức
UI2TOOL.Tooltip.showTooltip(element, "Nội dung tooltip", "right");

// Cú pháp rút gọn
UI2TOOL.Tooltip(element, "Tooltip rút gọn", "bottom");

// Ẩn tooltip
UI2TOOL.Tooltip.hide();

Điều khiển thủ công 

// Hiển thị tooltip thủ công
UI2TOOL.Tooltip(button, "Hiển thị tooltip này theo yêu cầu", "top");

// Ẩn tooltip cụ thể
UI2TOOL.Tooltip.hide();

Toast

Hiển thị thông báo toast với vị trí và kiểu tùy chỉnh.

Sử dụng cơ bản 

// Cú pháp đầy đủ
UI2TOOL.Toast.Show("Đây là thông báo toast");

// Cú pháp rút gọn
UI2TOOL.Toast("Đây là thông báo toast với cú pháp rút gọn");

Vị trí Toast 

// Toast ở vị trí cụ thể
UI2TOOL.Toast("Toast ở góc trên bên phải", { position: "top-right" });
UI2TOOL.Toast("Toast ở giữa màn hình", { position: "center" });
UI2TOOL.Toast("Toast ở góc dưới bên trái", { position: "bottom-left" });
UI2TOOL.Toast("Toast ở góc dưới bên phải", { position: "bottom-right" });
UI2TOOL.Toast("Toast ở giữa bên trái", { position: "center-left" });
UI2TOOL.Toast("Toast ở giữa bên phải", { position: "center-right" });
UI2TOOL.Toast("Toast ở góc trên bên trái", { position: "top-left" });
UI2TOOL.Toast("Toast ở góc dưới bên trái", { position: "bottom-left" });

Các Loại Toast 

// Các loại Toast
UI2TOOL.Toast("Thông báo thành công", { type: "success" });
UI2TOOL.Toast("Thông báo lỗi", { type: "error" });
UI2TOOL.Toast("Cảnh báo", { type: "warning" });
UI2TOOL.Toast("Thông tin", { type: "info" });

Toast từ Instance với Ghi đè Vị trí 

// Tạo instance với vị trí mặc định là bên trái
const UI_Left = new UI2TOOL('ui-left');

// Gọi với vị trí mặc định
UI_Left.Toast("Toast hiển thị ở vị trí mặc định bên trái");

// Ghi đè vị trí - hiển thị ở góc trên bên phải thay vì bên trái
UI_Left.Toast("Toast ghi đè vị trí sang góc trên bên phải", { position: "top-right" });

Alert

Hiển thị hộp thoại alert dạng modal với nội dung, tiêu đề và nút có thể tùy chỉnh. Khác với Toast, Alert không có tùy chọn vị trí.

Lưu ý: Nên chỉ sử dụng UI2TOOL.Alert mặc định cho modal. Không khuyến khích tạo nhiều instance Alert vì có thể gây ra các alert chồng lên nhau và gây nhầm lẫn cho người dùng.

Alert Cơ Bản 

// Chỉ hiển thị nội dung (tiêu đề mặc định là "Thông báo")
UI2TOOL.Alert("Đây là nội dung của alert");

// Cú pháp rút gọn
UI2TOOL.Alert("Đây là nội dung với cú pháp rút gọn");

Alert với Tiêu đề 

// Hiển thị với nội dung và tiêu đề tùy chỉnh
UI2TOOL.Alert.Show("Đây là nội dung của alert", "Tiêu đề tùy chỉnh");

Alert Nâng cao 

// Alert với nhiều tùy chọn nâng cao
UI2TOOL.Alert.Show({
    title: "Xác nhận thao tác",
    content: "Bạn có chắc chắn muốn thực hiện hành động này?
Lưu ý: Hành động không thể hoàn tác.",
    icon: '',
    buttons: [
        { 
            label: "Xác nhận", 
            className: "bg-blue-600 text-white px-4 py-2 rounded", 
            onClick: () => { 
                UI2TOOL.Toast("Đã xác nhận thao tác!", { type: "success" });
            }
        },
        { 
            label: "Hủy", 
            className: "bg-gray-500 text-white px-4 py-2 rounded", 
            close: true 
        }
    ],
    onClose: () => { 
        console.log("Alert đã đóng");
    }
});
